# Data-Analytics-Project-3
SQL Data Analysis Project

## Aim

The aim of this project is to use SQL queries to analyze data stored in a database. It focuses on retrieving, filtering, sorting, grouping, and summarizing information to gain useful insights from the dataset.

## Objective

- Create and use a MySQL database.
- Import the dataset into a table.
- Retrieve data using `SELECT` queries.
- Filter records using `WHERE`.
- Sort data using `ORDER BY`.
- Group data using `GROUP BY`.
- Calculate summary statistics using `COUNT()`, `SUM()`, and `AVG()`.
- Practice basic SQL skills for data analysis.

## Tools Used
- MySQL
- MySQL Command Line / MySQL Workbench
- SQL

## Tasks Performed
- Created a new database
- Created the `orders` table
- Imported the dataset into MySQL
- Retrieved data using `SELECT`
- Filtered records using `WHERE`
- Sorted records using `ORDER BY`
- Grouped data using `GROUP BY`
- Calculated total orders using `COUNT()`
- Calculated total revenue using `SUM()`
- Calculated average order value using `AVG()`
